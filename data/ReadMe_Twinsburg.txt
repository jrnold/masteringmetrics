** ReadMe_Twinsburg.txt **

This folder contains the data and programs used to generate Table 6.2 in
Mastering Metrics. Table 6.2 presents estimates of returns to schooling for the Twinsburg twins. These data are used in Ashenfelter and Krueger (AER 1994) and Ashenfelter and Rouse (QJE 1998). Data should be downloaded from http://arks.princeton.edu/ark:/88435/dsp01rv042t084.

Data: 
* Download “pubtwins.dta” from http://arks.princeton.edu/ark:/88435/dsp01rv042t084. Create a subdirectory titled “Data” in the same directory that contains your .do file. In the same directory, create another subdirectory titled “Output”. Save “pubtwins.dta” in the Data subdirectory.

Code: 
* /Twinsburg/Code/twins.do

Output: 
* /Twinsburg/Output/twins.xls (contains all of Table 6.2)
